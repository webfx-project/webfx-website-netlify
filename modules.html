Project Modules

This project has declared the following modules:

Name Description
webfx-platform-arch Cross-platform Arch API which provides information about the platform architecture.
webfx-platform-arch-gwt-j2cl GWT implementation of the Arch API (for the browser).
webfx-platform-arch-java Java implementation of the Arch API (for all platforms other than browser).
webfx-platform-ast The WebFX Platform is a set of low-level APIs designed for Java-based cross-platform development.
webfx-platform-ast-factory-generic The WebFX Platform is a set of low-level APIs designed for Java-based cross-platform development.
webfx-platform-ast-factory-gwt The WebFX Platform is a set of low-level APIs designed for Java-based cross-platform development.
webfx-platform-ast-factory-teavm The WebFX Platform is a set of low-level APIs designed for Java-based cross-platform development.
webfx-platform-ast-factory-vertx The WebFX Platform is a set of low-level APIs designed for Java-based cross-platform development.
webfx-platform-ast-json-plugin The WebFX Platform is a set of low-level APIs designed for Java-based cross-platform development.
webfx-platform-ast-yaml-plugin The WebFX Platform is a set of low-level APIs designed for Java-based cross-platform development.
webfx-platform-async Platform-independent API for asynchronous operations (such as the Fetch API provided by the webfx-platform-fetch module).
webfx-platform-audio Cross-platform Audio API. Most users will use the JavaFX Media API instead, which runs on desktops (via OpenJFX) and in the browser (via a GWT implementation provided by the WebFX Kit). Gluon doesn't support the JavaFX Media API, but the WebFX Kit does (for the audio) through mapping the JavaFX Media API to this Audio API (which has a Gluon implementation). A backward mapping is also supported for WebFX applications that may use this Audio API directly: in this case, the WebFX Kit will map this Audio API back to the JavaFX Media API for platforms other than mobiles.
webfx-platform-audio-gluon Gluon implementation of the Audio API (for mobiles).
webfx-platform-blob The WebFX Platform is a set of low-level APIs designed for Java-based cross-platform development.
webfx-platform-blob-gwt-j2cl The WebFX Platform is a set of low-level APIs designed for Java-based cross-platform development.
webfx-platform-blob-java The WebFX Platform is a set of low-level APIs designed for Java-based cross-platform development.
webfx-platform-boot Cross-platform API & SPI for booting WebFX applications. It defines a single entry point for all WebFX applications and sequences the application boot. If necessary, an application module such as a plugin module can be called during the boot sequence by implementing the SPI. An alternative SPI allows WebFX applications to start jobs at the end of the boot sequence.
webfx-platform-boot-java Java implementation of the Boot API (for all platforms other than browser and server).
webfx-platform-boot-vertx Vert.x implementation of the Boot API (for the server).
webfx-platform-browser The WebFX Platform is a set of low-level APIs designed for Java-based cross-platform development.
webfx-platform-browser-gluon The WebFX Platform is a set of low-level APIs designed for Java-based cross-platform development.
webfx-platform-browser-gwt-j2cl The WebFX Platform is a set of low-level APIs designed for Java-based cross-platform development.
webfx-platform-conf The WebFX Platform is a set of low-level APIs designed for Java-based cross-platform development.
webfx-platform-conf-file-java-plugin The WebFX Platform is a set of low-level APIs designed for Java-based cross-platform development.
webfx-platform-conf-zero The WebFX Platform is a set of low-level APIs designed for Java-based cross-platform development.
webfx-platform-conf-zero-impl The WebFX Platform is a set of low-level APIs designed for Java-based cross-platform development.
webfx-platform-console Cross-platform API for logging in the console.
webfx-platform-console-gwt-j2cl GWT implementation of the Console API (for the browser).
webfx-platform-console-java Java implementation of the Console API (for all platforms other than browser).
webfx-platform-console-teavm Java implementation of the Console API (for all platforms other than browser).
webfx-platform-fetch Cross-platform Fetch API (mock of the Web Fetch API) for fetching remote resources such as Json web services.
webfx-platform-fetch-ast-json The WebFX Platform is a set of low-level APIs designed for Java-based cross-platform development.
webfx-platform-fetch-gwt-j2cl GWT implementation of the Fetch API (for the browser).
webfx-platform-fetch-java Java implementation of the Fetch API (for platforms other than browser and server).
webfx-platform-fetch-vertx Vert.x implementation of the Fetch API (for the server).
webfx-platform-file Cross-platform API for accessing local files (mock of the Web File API). Files can't be created programmatically, but they can be instantiated by some other APIs such as the FilePicker provided as part of WebFX Extras. More info can be found in this <a href="https://blog.webfx.dev/2022/10/19/accessing-local-files/">post</a>.
webfx-platform-file-gwt-j2cl GWT implementation of the File API (for the browser).
webfx-platform-file-java Java implementation of the File API (for platforms other than the browser).
webfx-platform-javabase-emul-gwt Set of GWT super sources to emulate some JDK classes not emulated by default by GWT but required by WebFX.
webfx-platform-javabase-emul-j2cl The WebFX Platform is a set of low-level APIs designed for Java-based cross-platform development.
webfx-platform-javatime-emul-j2cl The WebFX Platform is a set of low-level APIs designed for Java-based cross-platform development.
webfx-platform-javatime-emul-tzdb The WebFX Platform is a set of low-level APIs designed for Java-based cross-platform development.
webfx-platform-meta Platform-independent Meta API for getting information about the WebFX application itself.
webfx-platform-os Cross-platform OS API for getting information about the host operating system.
webfx-platform-os-gluon Gluon implementation of the OS API (for mobiles).
webfx-platform-os-gwt-j2cl GWT implementation of the OS API (for the browser).
webfx-platform-os-java Java implementation of the OS API (for platforms other than browser and mobiles).
webfx-platform-reflect The WebFX Platform is a set of low-level APIs designed for Java-based cross-platform development.
webfx-platform-resource Cross-platform Resource API for accessing local resources.
webfx-platform-resource-gluon Gluon implementation of the Resource API (for mobiles).
webfx-platform-resource-gwt GWT implementation of the Resource API (for the browser).
webfx-platform-resource-j2cl The WebFX Platform is a set of low-level APIs designed for Java-based cross-platform development.
webfx-platform-resource-java Java implementation of the Resource API (for platforms other than browser).
webfx-platform-scheduler Cross-platform Scheduler API for scheduling non-UI tasks. The GWT implementation is indirectly provided by the UiScheduler API (an extension of the Scheduler API with additional methods for UI tasks), because there is no distinction in terms of implementation between these 2 APIs for the browser (due to its mono-thread nature).
webfx-platform-scheduler-java Java implementation of the Scheduler API (for platforms other than browser adn server).
webfx-platform-scheduler-vertx Vert.x implementation of the Scheduler API (for the server).
webfx-platform-service The WebFX Platform is a set of low-level APIs designed for Java-based cross-platform development.
webfx-platform-service-gwt-j2cl The WebFX Platform is a set of low-level APIs designed for Java-based cross-platform development.
webfx-platform-shutdown Cross-platform Shutdown API for shutting down WebFX applications.
webfx-platform-shutdown-gluon Gluon implementation of the Shutdown API (for mobiles).
webfx-platform-shutdown-gwt-j2cl GWT implementation of the Shutdown API (for the browser).
webfx-platform-shutdown-java Java implementation of the Shutdown API (for platforms other than browser and mobiles).
webfx-platform-storage Cross-platform Storage API for client-side storage.
webfx-platform-storage-gwt-j2cl GWT implementation of the Storage API (for the browser).
webfx-platform-storage-java Java implementation of the Storage API (for platforms other than the browser).
webfx-platform-storagelocation Cross-platform StorageLocation API for locating application internal file storage. There is no GWT implementation because file storage is not supported by browsers. This API is mainly designed for mobiles and desktops.
webfx-platform-storagelocation-gluon Gluon implementation of the StorageLocation API (for mobiles).
webfx-platform-storagelocation-java Java implementation of the StorageLocation API (for platforms other than mobiles & browser).
webfx-platform-substitution The WebFX Platform is a set of low-level APIs designed for Java-based cross-platform development.
webfx-platform-substitution-var The WebFX Platform is a set of low-level APIs designed for Java-based cross-platform development.
webfx-platform-substitution-var-conf-plugin The WebFX Platform is a set of low-level APIs designed for Java-based cross-platform development.
webfx-platform-substitution-var-env-plugin The WebFX Platform is a set of low-level APIs designed for Java-based cross-platform development.
webfx-platform-substitution-var-localstorage-plugin The WebFX Platform is a set of low-level APIs designed for Java-based cross-platform development.
webfx-platform-substitution-var-sysprop-plugin The WebFX Platform is a set of low-level APIs designed for Java-based cross-platform development.
webfx-platform-substitution-var-windowlocation-plugin The WebFX Platform is a set of low-level APIs designed for Java-based cross-platform development.
webfx-platform-typedarray The WebFX Platform is a set of low-level APIs designed for Java-based cross-platform development.
webfx-platform-typedarray-gwt-j2cl The WebFX Platform is a set of low-level APIs designed for Java-based cross-platform development.
webfx-platform-uischeduler Cross-platform UiScheduler API for scheduling UI tasks. It's an extension of the Scheduler API with additional methods for handling UI tasks. The WebFX Platform provides only the GWT implementation. The WebFX Kit provides the OpenJFX implementation.
webfx-platform-uischeduler-gwt-j2cl GWT implementation of the UiScheduler API (for the browser).
webfx-platform-useragent Cross-platform UserAgent API for getting information about the user agent.
webfx-platform-useragent-gluon Gluon implementation of the UserAgent API (for mobiles).
webfx-platform-useragent-gwt-j2cl GWT implementation of the UserAgent API (for the browser).
webfx-platform-useragent-java-client Java client-side implementation of the UserAgent API (for desktops).
webfx-platform-useragent-java-server Java server-side implementation of the UserAgent API (for servers).
webfx-platform-util A set of utility classes.
webfx-platform-util-http The WebFX Platform is a set of low-level APIs designed for Java-based cross-platform development.
webfx-platform-util-time The WebFX Platform is a set of low-level APIs designed for Java-based cross-platform development.
webfx-platform-vertx-common A set of common classes used by Vert.x implementation modules.
webfx-platform-visibility Cross-platform Visibility API to know and react to the application visibility state.
webfx-platform-visibility-gluon Gluon implementation of the Visibility API (for mobiles).
webfx-platform-visibility-gwt-j2cl GWT implementation of the Visibility API (for the browser).
webfx-platform-visibility-java-headless Java headless implementation of the Visibility API.
webfx-platform-webassembly Cross-platform WebAssembly API for working with WebAssembly.
webfx-platform-webassembly-teavm TeaVM implementation of the WebAssembly API.
webfx-platform-webworker Cross-platform WebWorker API for working with web workers.
webfx-platform-webworker-gwt-j2cl GWT implementation of the WebWorker API (for the browser).
webfx-platform-webworker-java Java implementation (based in Java Threads) of the WebWorker API (for platforms other than the browser).
webfx-platform-webworker-teavm TeaVM implementation of the WebWorker API (for the browser).
webfx-platform-windowhistory Cross-platform WindowHistory API for accessing the window history.
webfx-platform-windowhistory-gwt-j2cl GWT implementation of the WindowHistory API (for the browser).
webfx-platform-windowhistory-java Java implementation of the WindowHistory API (for platforms other than the browser).
webfx-platform-windowhistory-web Abstract Web implementation of the WindowHistory API (for the browser).
webfx-platform-windowlocation Cross-platform WindowLocation API for accessing the window location.
webfx-platform-windowlocation-gwt-j2cl GWT implementation of the WindowLocation API (for the browser).
webfx-platform-windowlocation-java Java implementation of the WindowLocation API (for platforms other than the browser).
